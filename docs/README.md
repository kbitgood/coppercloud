# coppercloud docs

This directory contains the longer-form documentation for `coppercloud`.

## Guides

- [`getting-started.md`](./getting-started.md): installation, first-run setup, and the fastest path to a successful request
- [`command-reference.md`](./command-reference.md): every command, option, behavior note, and help example
- [`examples.md`](./examples.md): practical recipes for common API workflows
- [`ai-agents.md`](./ai-agents.md): guidance for using `coppercloud` in agent-driven and automation-heavy environments

## Documentation goals

The docs are designed to be useful in three places:

- The npm package page, where the root `README.md` should explain the tool quickly
- The terminal, where `--help` should answer the next question without sending someone to a website
- The repository docs, where examples and edge cases can be explained in more detail
